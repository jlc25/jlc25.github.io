Context

This project was a group effort at learning Kotlin and mobile app development for Android in particular.

Introduction

Ribbitivity welcomes the user warmly with its productivity frog. The user is able to write down what tasks they want to complete and use a timer to frame intervals of productivity (inspired by the pomodoro technique)

Challenges

Our biggest challenge in this project was that we had no prior knowledge regarding the development of mobile apps.
